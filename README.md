# UpSync: Remote File Sync Tool
UpSync is a tool that allows a user to update one or more remote files with their local device's version in a single click. 

## About This App
UpSync was designed to address an issue where many applications cannot save directly to legacy Kerberos network shares on autopilot devices. It is intended for use by teams who routinely output files to specific locations as part of their workflows. 

## How It Works
UpSync monitors a local cache folder on your device. Each file within the cache can be associated with a remote filepath of your choosing. 

When you update your local copy of any files, UpSync compares them with their remote counterparts, showing whether they are in sync. 

A quick click of the Sync button makes the remote copy match your local copy. You can sync individual files or many at once with Sync All. 

## Planned Features:
- Visible Sync status between local and remote files (Local is newer / In sync / Remote is newer)
- See last user to update remote file
- Built-in automatic backup of files
- Ability to restore file to previous state
- Exportable configuration for sharing your settings with your team
